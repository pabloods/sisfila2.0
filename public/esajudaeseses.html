1. Como inserir uma grande quantidade de alunos no sistema?

Primeiramente vá até a área de trabalho e crie uma pasta chamada "matricula-ccc";
Após criar a pasta insira os 3 arquivos listados abaixo:

O primeiro arquivo deve conter o escalonamento dos alunos (scores) e deve se chamar escalonamento-alunos.txt;
O segundo arquivo deve conter os formandos e deve se chamar formandos.txt;
O terceiro arquivo deve conter os alunos que fizeram pré matrícula e deve se chamar prematricula.csv;

# Caso o sistema esteja rodando localmente (na sua própria maquina), utilize os comandos:
FILE=~/Desktop/matricula-ccc/escalonamento-alunos.txt rails runner script/le-escalonamento.rb FILE=~/Desktop/matricula-ccc/formandos.txt rails runner script/le-formandos.rb FILE=~/Desktop/matricula-ccc/prematricula.csv rails runner script/le-prematricula.rb

# Caso o sistema esteja hospedado no Heroku, utilize os seguintes comandos: cat ~/Desktop/matricula-ccc/escalonamento-alunos.txt | heroku run --no-tty FILE=/dev/stdin rails runner script/le-escalonamento.rb cat ~/Desktop/matricula-ccc/formandos.txt | heroku run --no-tty FILE=/dev/stdin rails runner script/le-formandos.rb cat ~/Desktop/matricula-ccc/prematricula.csv | heroku run --no-tty FILE=/dev/stdin rails runner script/le-prematricula.rb

2. Como iniciar a chamada de alunos?

Primeiramente você deve certificar que existam alunos na fila, para se certificar vá até "Vagas" e veja se já existe algum aluno na fila.

Caso a fila já possua alunos você deve escolher a mesa em que deseja ocupar. Vá até "Mesas" e selecione uma mesa clicando em "Escolher". Caso o sistema não possua nenhuma mesa cadastrada então crie uma, clicando em "Criar Nova Mesa".

Após selecionar a mesa vá até "Colegiados" e selecione o colegiado em que deseja realizar as chamadas clicando em "Mostrar". Caso o colegiado não exista, clique em "Criar Novo Colegiado".
